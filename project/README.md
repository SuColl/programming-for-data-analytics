# Arts Council Funding by Artform, 2003-2017  
This repository explores the funding awarded to individual artists by the Irish 
Arts Council, and analyses funding trends by artform.

## Data Source  
- CSO publications on "Women and Men in Ireland", [link](https://www.cso.ie/en/statistics/womenandmeninireland/).   
(Note: 2013 publication does not contain any Arts Council data.)  
**Note: the date in the URL refers to the date of publication, not the period of the data.**

## Requirements
- Python 3+
- Data manipulation
  - [NumPy](https://numpy.org/doc/stable/ ) Array manipulation 
  - [Pandas](https://pandas.pydata.org/docs/) Python Data Analysis Library
- Data Visualisation
  - [Matplotlib.PyPlot](https://matplotlib.org/stable/api/pyplot_summary.html) for plotting
